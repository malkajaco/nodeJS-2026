CORS

no se pueden hacer peticiones de un sitio a otro, por eso se usa CORS. Se puede habilitar en el backend con un middleware. En este caso, se está haciendo una petición desde un archivo HTML a un servidor que corre en localhost:3000. Si el servidor no tiene habilitado CORS, la petición será bloqueada por el navegador. si es una api para una app de celular no hace falta porque no se va a usar
